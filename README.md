# URL Shortener

A scalable URL shortener built with FastAPI, PostgreSQL, and Redis.

## Features
- Short URL generation
- Redirect to original URL
- URL expiry handling
- Click tracking and analytics
- REST API design

## Tech Stack
- Python
- FastAPI
- PostgreSQL
- Redis
- SQLAlchemy
- Docker

## Project Goals
This project is built to demonstrate backend engineering skills including API development, database design, caching, and scalable system design.

## Planned Endpoints
- Create short URL
- Redirect short URL
- Get URL analytics
- Set expiry time for links
